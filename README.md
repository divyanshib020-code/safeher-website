# SafeHer – Women Safety Web Application

SafeHer is a web-based project focused on women’s safety. The idea is to create a quick and simple way for users to send an SOS alert with their live location during emergency situations.

The main goal of this project is to reduce the time and steps needed to ask for help. Instead of going through multiple apps or options, this system is designed to work in a fast and straightforward way.

Features:
- One-tap SOS alert system
- Share live location with emergency contacts
- Add and manage trusted contacts
- Secure login with password protection
- Clean and easy-to-use interface

Problem it solves:
In many emergency situations, people don’t get enough time to unlock their phone, search for an app, and send help. Existing solutions can be slow or complicated. SafeHer tries to solve this by providing a quick and simple SOS system.

Tech used:
- HTML
- CSS
- JavaScript
- PHP
